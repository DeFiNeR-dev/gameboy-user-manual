# Playing Games on the Game Boy Color

<img
  style={{ width: 800 }}
  src="/gameboy-user-manual/img/video-game-system.png"
/>

1. Insert a Game Pak into the Game Pak slot at the back of the Game Boy Color. The label should face away from the Game Boy Color.
2. Turn on the power by pushing the Power Switch upwards. Be sure to read the Game Pak instructor booklet for the game you are playing.
3. To remove the Game Pak after playing, first turn the power OFF by sliding the Power Switch downward, then slide the Game Pak out of the Game Pak Slot.

:::caution
Always check that the Game Pak Slot is clean before inserting the Game Pak.
:::
